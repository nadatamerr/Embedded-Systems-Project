Auto Car Parking System 
ReadME file

Link to Wiki Page: https://github.com/nadatamerr/Embedded-Systems-Project/wiki/G7:-Car-Auto-Parking-System 

Steps to run the code:

1. Connect the STM microcontroller to the PC and load the project onto it using Keil uVision IDE (and reset the microcontroller). 
2. Connect the microcontroller, the 3 sensors, the bluetooth controller and the Dagu car as shown in the schematic diagram in the wiki.
3. Download Arduino Blue mobile application and connect it to the bluetooth controller.
4. Turn the car switch on.
5. Send "S" using the bluetooth command. The car should move forward, searching for a parking spot.
